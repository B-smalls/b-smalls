# Backend Developer (Go / Python)

Привет! Я Никита, backend-разработчик. Работаю с микросервисными системами на **Go** и **Python**: проектирую REST API, реализую сервисную логику, интеграции между системами и асинхронные сценарии через брокеры сообщений.

Мне интересен backend не только как набор эндпоинтов, а как инженерная система: контракты между сервисами, обработка ошибок, транзакции, очереди, события, хранение данных и устойчивость всей цепочки.

Основной стек: **Go, Python, PostgreSQL, MongoDB, RabbitMQ, Docker**.

В работе стараюсь делать решения понятными и поддерживаемыми: с читаемой структурой, предсказуемой архитектурой и логикой, которую можно развивать без лишнего хаоса.
